Как это работает
Beget в этом случае не хостит Next.js — он только отвечает за запись DNS «поддомен → Vercel». Бесплатный виртуальный хостинг для этого подходит: платить за VPS не нужно.

Пошагово
1. Придумайте поддомен
Например: reviews.ваш-домен.ru или rent.ваш-домен.ru.

Домен должен быть привязан к Beget (зарегистрирован там или делегирован на их NS).

2. В Vercel (у вас уже открыт раздел Domains)
Нажмите Add Existing
Введите полный поддомен: reviews.ваш-домен.ru
Vercel покажет, какую DNS-запись добавить — обычно для поддомена это CNAME
Типичное значение:

Тип	Имя / Host	Значение
CNAME
reviews (или полное имя — как просит Beget)
cname.vercel-dns.com
Точное значение берите из панели Vercel — оно может отличаться.

3. В панели Beget
Домены → ваш домен → DNS / Управление DNS
Добавьте CNAME-запись для поддомена
Сохраните
Поддомен в разделе «Сайты» Beget создавать не обязательно, если трафик идёт на Vercel. Нужна только DNS-запись.

4. Подождите и проверьте
DNS обновляется от нескольких минут до 24–48 часов (часто 15–30 минут)
В Vercel статус сменится на Valid Configuration
SSL (HTTPS) Vercel выдаст сам
Важные нюансы
Бесплатный Hobby на Vercel — свои домены поддерживает, для личного проекта этого достаточно.

Корневой домен (ваш-домен.ru без поддомена) на Beget иногда сложнее: у apex нельзя CNAME, нужны A-записи на IP Vercel. Поддомен (reviews.…) — проще, через CNAME.

Если домен не на Beget — DNS можно настроить там, где домен реально обслуживается (Reg.ru, Cloudflare и т.д.). Beget нужен только если NS домена указывают на Beget.

www — если нужен www.reviews.…, его добавляют отдельно в Vercel и отдельной DNS-записью.

Что сделать
1. Vercel → Domains → Add Existing
Добавьте: reviews.bginfo.eu

Vercel покажет DNS-запись — обычно так:

Тип	Имя	Значение
CNAME
reviews
cname.vercel-dns.com
(Точное значение смотрите в панели Vercel.)

2. Beget → DNS для bginfo.eu
Добавьте CNAME:

Имя: reviews (иногда панель просит полное reviews.bginfo.eu)
Значение: cname.vercel-dns.com
На Beget не включайте отдельный SSL для этого поддомена и не создавайте сайт в хостинге — только DNS.

3. Подождите
Через 15–60 минут (иногда до суток) в Vercel появится Valid Configuration, и https://reviews.bginfo.eu заработает с зелёным замком.

Когда SSL на Beget нужен
Только если сайт физически крутится на Beget (PHP/статика). У вас сайт на Vercel — SSL там, не на Beget.

Если HTTPS не поднялся
В Vercel домен ещё не Valid — подождите или проверьте CNAME.
На Beget для reviews включён хостинг/прокси — отключите, оставьте только DNS.
Старый A-запись на reviews конфликтует с CNAME — удалите A, оставьте CNAME на Vercel.
Итого: настраиваете только DNS reviews → Vercel, SSL ставить не нужно.
